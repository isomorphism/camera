camera-0.0.3.0: Generate transformation matrices for a virtual camera

This package provides a wide assortment of ways to control the properties of a virtual camera and then generate the appropriate projection matrix. It allows adjusting the "focal length", setting the viewport dimensions/aspect ratio and automatically adjusting the projection to match, orbit the camera around a target point, &c.

The primary API for this package relies on lens, but usage examples are provided for anyone unfamiliar with it.
