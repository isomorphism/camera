data CoordinateSystem a

Describes the coordinate system of an unrotated camera.

Constructors

VerticalZ

Coordinate system common in 3D modeling applications. The positive Z axis points up and the positive Y axis points forward.

ZForward

Coordinate system common in 3D graphics programming. The positive Y axis points up and the positive Z axis points forward. This is a left-handed coordinate system.

ZBackward

Coordinate system common in 3D graphics programming. The positive Y axis points up and the positive Z axis points backward. This is a right-handed coordinate system.

OtherCoords (M33 a)

Arbitrary coordinate system, specified as (V3 rightward upward forward)
